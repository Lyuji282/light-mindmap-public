# Light Mindmap

A lightweight, fast mindmap editor built with Tauri.

## Download

Go to [Releases](../../releases) to download the latest build for your platform:

- **Windows**: `.msi` / `.exe`
- **macOS**: `.dmg`
- **Linux**: `.deb` / `.AppImage`

## Features

- Fast, native desktop app
- Intuitive mindmap editing
- Multiple node types
- Import / Export support
